#Vallaservice

Swedish website for a ski company.

#Build

Requires Node

##Install build dependecies

```CLI
$ npm install
```

##Build html

```CLI
$ node_modules/.bin/cake build:html > index.html
```

##Compile less

```CLI
$ node_modules/.bin/cake build:css > style.css
```
